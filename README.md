# AI-Engineer

Bonjour, ce git répertorie les projets de mon cursus OpenClassroom `AI Engineer`

## Structure des projets
Chaque projet est inclus dans un répertoire annoté du numéro de projet `pn`.
Par exemple pour le projet 2, accédez au dossier `p2`.

**Pour chaque projet vous retrouverez une notice d'utilisation :**

* `README.md`

## Docker

Ici nous avons fait le choix dans un premier temps d'utiliser `docker`. 
Chaque projet utilisant `docker` contient ses fichiers de build Docker : 

* `docker-compose.yaml`
* `Dockerfile` si nécessaire
* etc.

**Vous retrouverez une notice d'utilisation dans chaque projet.**
